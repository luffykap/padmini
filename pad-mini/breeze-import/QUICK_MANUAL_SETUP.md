## ðŸ’¡ **Pro Tips**

### **1. Use Task Templates**
Create a template task with standard fields:
- Description format
- Standard tags
- Time estimates
- Then duplicate and edit

### **2. Bulk Add via Copy-Paste**
Some Breeze views let you paste multiple task names:
```
Task 1
Task 2
Task 3
```
Then add details one by one.

### **3. Import from GitHub Issues**
If you have GitHub:
- Create issues from CSV
- Link Breeze to GitHub (if supported)
- Import automatically

### **4. Use Keyboard Shortcuts**
- `N` or `+` to add new task (in many project tools)
- `Enter` to save
- Speed up data entry
